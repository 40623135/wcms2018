基本功能

動態功能:

以 python wsgi.py 啟動

在瀏覽器中管理網際檔案內容

利用瀏覽器上傳檔案 (可同時上傳多個檔案)

檔案包含一般常用數位檔案與影像檔案

使用者並可直接在瀏覽器的頁面編輯區中引用檔案或顯示圖形檔案

靜態網頁:

在近端以 python -m http.server 執行, 或在 Github Pages 模式下執行.
